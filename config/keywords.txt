# TrendPulse - 关键词订阅配置
#
# 语法说明：
#   - 每行一个关键词组，同一行的词用空格分隔
#   - 普通关键词：直接写词，如 "AI"（OR 匹配，命中任一即推送）
#   - 必须词：+前缀，如 "+手机"（AND 匹配，必须同时命中）
#   - 过滤词：!前缀，如 "!广告"（排除匹配，命中则跳过）
#   - 数量限制：@数字，如 "@5"（限制该组最多显示5条）
#   - 正则表达式：/pattern/，如 "/\bAI\b/"
#   - 注释：# 开头的行会被忽略
#   - 留空则推送所有热点（不做过滤）
#
# 示例：
#   AI 人工智能 ChatGPT                → 包含任一词即推送
#   华为 苹果 +手机 !水果              → 必须含"手机"，排除含"水果"
#   特斯拉 马斯克 @5                   → 最多推送 5 条
#   /\bGPT\b/ 大模型                   → 精确匹配 GPT 单词
#
# [GLOBAL_FILTER] 区域：全局过滤，优先级最高
# [WORD_GROUPS] 区域：关键词组
